# Code of Conduct

MissionCache adopts **[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct. The canonical text at that link is authoritative; please read it before participating in the project.

## Reporting

To report a code-of-conduct concern:

- **Public / non-sensitive**: open a GitHub issue with the `conduct` label at https://github.com/missioncache/missioncache/issues
- **Private / sensitive**: use GitHub's private reporting channel at https://github.com/missioncache/missioncache/security - this is the closest private contact channel missioncache has until a dedicated one exists

MissionCache is a single-maintainer project. Response times are best-effort, but every report is read personally.
